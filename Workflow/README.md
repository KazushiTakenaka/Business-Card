# Workflow ディレクトリ

このディレクトリには、名刺管理システムの各種ワークフローに関するドキュメントが含まれています。

## ワークフロー一覧

### 1. [PDF名刺情報抽出ワークフロー](pdf_extraction_workflow.md)

Google Drive上の名刺PDFから情報を抽出してGoogleスプレッドシートに登録する完全なワークフローです。

**使用シーン:**
- 新しい名刺PDFをスプレッドシートに追加する
- 大量の名刺PDFを一括処理する

**主なステップ:**
1. PDFをダウンロード
2. Claude Codeで名刺情報を抽出・登録
3. クリーンアップ（ローカル削除・Drive移動）

---

## ワークフロー実行の前に

すべてのワークフローを実行する前に、以下を確認してください：

1. **環境準備**
   ```bash
   .venv\Scripts\activate
   ```

2. **設定確認**
   - `PyPrograms/config.py`の設定が正しいこと
   - サービスアカウントキーが有効であること
   - 必要な権限が付与されていること

3. **依存関係**
   ```bash
   pip install -r requirements.txt
   ```

新しいワークフローを追加する際は、以下のテンプレートを参考にしてください：

```markdown

[ワークフローの簡単な説明]

## 前提条件
- [必要な準備]

## ステップ

### ステップ1: [タイトル]
[詳細な説明]

### ステップ2: [タイトル]
[詳細な説明]

## トラブルシューティング
[よくある問題と対処法]

## 関連ファイル
- [関連するスクリプトやファイル]
```
